# CheatSheet: C

## Memory
__Stack:__ 
    - attached to thread 
    - static size
    - faster than heap due to trivial access pattern (increment pointer)
__Heap:__  
    - allocated on app startup
    - dynamic size 
    - more complex 'bookkeeping'
